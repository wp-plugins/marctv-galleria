== Description ==

Replaces the gallery code with a neat sliding image gallery. Based on the mighty 
Galleria http://galleria.io/ it comes with responsiveness and touch events! 
Includes Google Analytics Events, too!

== Installation ==

* Install plugin
* Activate it
